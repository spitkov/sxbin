# 🚀 sxbin - Simple File Sharing and Pastebin Service

sxbin is a versatile file sharing and pastebin service that offers a range of features for easy content sharing and management.

## 🌟 Key Features

- 📁 File Uploading: Easily upload and share files of any type, including HTML files that can be rendered directly in the browser
- 📝 Pastebin: Share text snippets and code
- 🔗 URL Shortener: Create short links for long URLs
- 🔐 Password Protection: Secure your uploads with passwords
- 👤 User Accounts: Create dedicated user pages

## 🔒 Password Protection

- Add passwords to your uploads for enhanced security
- Access password-protected content by appending the password to the URL:
  `https://sxbin.example.com/image.png/your-password`

## 👤 User Pages

- Create a personalized user page to manage all your uploads
- Organize your content with folders and custom layouts
- Share your public profile with others

## 🎨 Customization

- Customize your user page layout

## 🚀 Getting Started

1. Clone the repository
2. Install dependencies
3. Run the app with `python app.py`
